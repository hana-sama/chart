### How to read the table  

* **Dots** – the six‑dot positions (1‑6) used on a Braille cell.  
* **Braille Pattern** – the actual printable Unicode character you can copy‑paste.  
* **Unicode** – the code‑point (e.g., `U+282F`).  

When you encounter a **Grade 2** text, the reader’s brain applies the contraction rules automatically; the same Unicode characters can be used for both visual display and tactile embossing.  

Feel free to extend the list with any additional contractions you need – the pattern is always “add dot 3 for a capital, add dot 6 for a number sign, and combine the base cells as shown above.” Happy contracting!
