## UEB基礎マッピング

```
a ⠁  b ⠃  c ⠉  d ⠙  e ⠑  f ⠋  g ⠛  h ⠓  i ⠊  j ⠚
k ⠅  l ⠇  m ⠍  n ⠝  o ⠕  p ⠏  q ⠟  r ⠗  s ⠎  t ⠞
u ⠥  v ⠧  w ⠺  x ⠭  y ⠽  z ⠵

数字（j=0, a=1, b=2, ...）
1⠁ 2⠃ 3⠉ 4⠙ 5⠑ 6⠋ 7⠛ 8⠓ 9⠊ 0⠚
```

---

## 使い方

| 操作 | 動作 |
|------|------|
| `F` `D` `S` `J` `K` `L` 同時押し | ドットを組み合わせ |
| `Space` | 現在のドットを確定 |
| `Backspace` | 直前の文字を削除 |
| `Escape` | 現在の入力をクリア |

---

## 動作確認ポイント

1. **例: 「hello」を入力**
   - `J` → `h` ⠓
   - `F` + `J` → `e` ⠑
   - `D` + `J` + `L` → `l` ⠇
   - `D` + `J` + `L` → `l` ⠇
   - `J` + `K` → `o` ⠕

2. **点字表示**: Unicode点字が直接エディターに表示されます
